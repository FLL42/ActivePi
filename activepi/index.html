The ActivePi is a handheld Raspberry Pi-powered device designed to get people physically active and teach them how to program in Python! Users can code exercise games and projects. We are making lesson plans to teach users how to code the ActivePi and allow them to code anything they want! For example, users can code a game in which the more you walk, the more you level up and you can unlock things such as power-ups and more advanced coding projects, so you develop physical fitness and computer programming skills in parallel. Created by Alan Wong, Ben Goldberg, Joshua Smith, Kunal Toomu, and Yash Gupta from FLL 42, Pythonian Rabbotics.
